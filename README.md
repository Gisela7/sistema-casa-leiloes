# Nome do Projeto: LeiloesTDSat

## Descrição do Projeto
Este projeto consiste em um sistema informatizado desenvolvido para gerenciar as rotinas e fluxos operacionais de uma casa de leilões, garantindo maior controle e eficiência no cadastro de lotes, lances e finalização de pregões.

## Tecnologias Utilizadas
* Java
* MySQL
* Git & GitHub
